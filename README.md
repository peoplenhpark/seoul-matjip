# 🍽️ 서울 4대문 안 맛집 체크리스트

광화문·서촌·북촌·명동·을지로의 진짜 맛집 30곳 체크리스트 PWA 앱.

## 📍 라이브
👉 **https://peoplenhpark.github.io/seoul-matjip/**

## ✨ 기능
- **30곳 맛집** 6개 카테고리 (양식·한식·일식·중식·동남아·유럽식)
- **지도 + 핀** · 핀치 줌 · ⛶ 크게보기 (사이드바)
- **방문 체크** 날짜·★평점·메모 기록 (localStorage)
- **🎲 룰렛** · **5개 추천 코스** · **➕ 내 맛집 추가**
- **빠른 실행** 📞 전화 · 🗺️ 카카오맵 · 📅 캐치테이블
- **영업 상태 뱃지** 🟢 영업 중 / ⏰ 30분 후 마감 / ⚫ 종료
- **PWA** 홈 화면 설치 · 오프라인 지원

## 🗂️ 파일 구성
| 파일 | 역할 |
|---|---|
| `index.html` | 루트 진입점 (redirect + OG 태그) |
| `seoul_restaurants.html` | 메인 앱 (30곳 데이터 + 지도 + JS 전체) |
| `seoul_4daemun.html`, `seoul_4daemun.svg` | 📜 상세 지도 (고지도 스타일, 별도 페이지) |
| `manifest.json` | PWA 매니페스트 |
| `sw.js` | Service Worker (오프라인 캐싱) |
| `icon.svg` | 앱 아이콘 (홈 화면) |
| `og-image.png` / `og-image.svg` | 카카오톡 공유 미리보기 이미지 |
| `og-png-convert.html` | OG 이미지 PNG 재생성 도구 (선택) |
